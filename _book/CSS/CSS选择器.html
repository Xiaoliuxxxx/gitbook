1.CSS选择器

1. id选择器

选择特定id的HTML元素:

    #app {
        color:red;
    }

2. class选择器

选择应用该class的HTML元素

    .center{
       color:red;
       text-align:center;
    }

3. 通配符选择器

这个选择器是匹配页面的所有元素，一般用来清除浏览器的默认样式

    *｛margin:0;padding:0｝

4.元素选择器

选择指定类型的HTML元素

    div{
        background-color:red;
    }

5.属性选择器

选择拥有该属性的特定元素,如选择所有拥有title属性的HTML元素

    [title]{
        color:red;
    }

可以结合元素选择器精确选择目标，如

    /* 用来选择所有a标签拥有title属性的元素 */
    a[title]{
        color:red;
    }
    /* 用来选择img标签拥有alt属性的元素*/
    img[alt]{
        border:5px solid red
    }
    /*用来选择属性为具体某个值的元素*/
    a[href="www.baidu.com"]{
        color:red
    }
    /*用来选择属性包含某个值的元素*/
    a[href~="www.baidu"]{
        color:red
    }
    /*用来选择属性等于或以某个值开头的元素*/
    a[href|="www.baidu.com/"]{
        color:red
    }
    /*用来选择同时满足多个属性的元素*/
    a[href~="www.baidu.com"][title="baidu"]{
        color:red
    }

6.后代选择器和子代选择器

```css
/*后代选择器*/
    div p{
        color:red;
    }
/*子代选择器*/
    div>p｛
        color:red;
    ｝
```

区别：后代选择器父元素里所有符合的元素，子代选择器只选择符合内容的子元素（父元素下第一层）

7.相邻兄弟选择器

如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器

```css
/*紧接在div后的table元素选择*/
div + table {
    margin-left:20px
}
```

8.伪类

CSS伪类用于向某些选择器添加特殊效果

用法：

    a:hover{
        color:red;
    }
    input:focus{
        background-color:yellow;
    }

9.伪元素

用法：

```css
/*伪元素可以在元素的内容前面插入新内容*/
    h1:before{
      content:url(logo.gif);
    }
/*伪元素可以在元素的内容之后插入新内容*/
    h1:after{
      content:url(logo.gif);
    }
```

总结一下伪类与伪元素的特性及其区别：

  1. 伪类本质上是为了弥补常规CSS选择器的不足，以便获取到更多信息；
  2. 伪元素本质上是创建了一个有内容的虚拟容器；
  3. CSS3中伪类和伪元素的语法不同；
  4. 可以同时使用多个伪类，而只能同时使用一个伪元素；

总结：CSS选择器的内容大概这些，后续更新会补充进去。
